Accessibility

✅
集約 aria-label： 「205系統 市役所前ゆき、あと5分、14時23分発、通常運行」を行全体で 1 つの読み上げに。
✅
状態変化： aria-live="polite"、arriving は assertive も可。
✅
取消線は色だけに頼らない： <s> + 「変更前」aria-label。
✅
等幅数字： tabular-nums で桁が揃い、視線移動だけで読める。
