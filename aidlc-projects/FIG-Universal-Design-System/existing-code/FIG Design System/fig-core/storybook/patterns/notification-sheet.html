Accessibility

✅
Sheet： role="dialog" + aria-modal="true" + aria-labelledby。
✅
集約 aria-label： 「未読、警告、205系統が遅延しています、14時23分発、5分前」を 1 行に統合。
✅
既読化は IntersectionObserver で 2 秒観測（誤タップでの全消失防止）。
✅
フォーカストラップ： Sheet 内に閉じ込め、閉じた瞬間に Bell へ戻す。
✅
Drag Handle 視覚 32×4 / 操作領域 44×44。

Motion

EventToken
Sheet 出現 --motion-modal-enter (280ms / decelerate)
Sheet 消滅 --motion-modal-exit (240ms / accelerate)
新着行 出現 --motion-enter + 下から slide
既読化 --motion-tab-switch bg crossfade
削除 swipe 右 slide-out + max-height: 0 同時
Bell badge 増加 scale 1.0→1.3→1.0 200ms（増加時のみ）
禁止： 自動スクロール（ユーザー操作を奪う）、color flash、Bell badge 無限振動。
