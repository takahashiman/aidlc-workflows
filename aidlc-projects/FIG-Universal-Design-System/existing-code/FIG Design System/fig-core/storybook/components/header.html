Accessibility

✅
左右スロットは Landmark で囲わない。<header role="banner"> 1 つだけ。中の button は単なる <button>。
✅
Hamburger: aria-label="メニューを開く" + aria-expanded + aria-controls={drawerId}。
✅
Region: aria-label="地域を切替（現在: 京都）" ── 視覚で省略している「県」を読み上げで補う。
✅
Title: <h1> はページ全体で 1 つに保つ（下層では非表示の sr-only に切替えるか、別文言にする）。
⚠
frosted bg のコントラスト： 背景がティール寄りでも文字 #334155 は AA を維持。背景がさらに暗い画面では `bg: rgba(255,255,255,0.85)` に上げる。
