Accessibility

✅
最小サイズ： モバイル 12px、推奨 14px 以上。caption (12px) は補助情報のみ。
✅
Contrast： 全 text color token は背景に対し AA (4.5:1) 以上。--color-text-muted は 14px 以上で使用。
✅
等幅数字： 時刻 / 運賃 / 系統番号は tabular-nums で桁揃え、視線の上下移動だけで読める。
✅
言語属性： 日本語ページは <html lang="ja">、数字混在時は <span lang="en"> でフォント切替が正しく働く。

Motion

テキスト自体はモーションを持たない。テキストを含む要素（Pill / Card / Banner）の motion は親コンポーネント側の責務。

禁止： 文字の text-shadow アニメ、color hue rotation、letter-spacing breathing。意味の不安定化を招く。
