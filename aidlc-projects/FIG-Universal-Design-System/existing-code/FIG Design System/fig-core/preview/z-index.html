Z-Index Layers

100 刻みの推奨スケール（--z-base: 0 〜 --z-tooltip: 700）をアイソメトリックに俯瞰したもの。各層の間に 99 個の差し込み余地を残してあります。コンポーネントは z-index を数値で直書きせず、必ず本トークン経由で指定します。

--z-base0
CONTENT
通常フロー、本文、リストセル、入力
--z-raised100
CARD
静止カード、低位ボタン（elevation.1）
--z-sticky200
FIXED
sticky ヘッダー、フッターナビ
--z-dropdown300
POPOVER
プルダウン、メニュー、ツールチップ予備
--z-fab400
FLOATING
FAB、スナックバー、固定 CTA
--z-drawer500
SHEET
サイドバー、ボトムシート
--z-modal600
DIALOG
モーダル、ダイアログ（背面 scrim 含む）
--z-toast / --z-tooltip700
TOPMOST
トースト、最前面通知、ツールチップ
