STATE TOKENS

hover / pressed / focused / disabled / dragged / selected

インタラクティブ要素に重ねる状態トークン。半透明オーバーレイ方式を基本とし、どの色のサーフェス上でも一貫した「触感」を出します。

state.hover

ポインタ接触時のフィードバック。ベース色の上にスレートのオーバーレイを重ねます。
サイドバー行（hover）
overlay: rgba(15,23,42,0.04)
brand-overlay: rgba(38, 183, 188,0.08)
bg (solid): #F8FAFC

state.pressed

押下中。scale 0.98 + 暗オーバーレイ。NFC など特殊操作は 0.92。
scale: 0.98（CTA） / 0.92（NFC）
overlay: rgba(15,23,42,0.10)
brand: rgba(38, 183, 188,0.16)

state.focused

キーボードフォーカス。3px のソフトリング。色は accent（indigo）。
ring: 0 0 0 3px rgba(56, 161, 219,0.40)
ring-brand: 0 0 0 3px rgba(38, 183, 188,0.30)
outline-offset: 2px

state.disabled

操作不可。opacity 40%、cursor not-allowed、ポインタイベント遮断。
opacity: 0.40
bg: #F1F5F9 / fg: #94A3B8
cursor: not-allowed

state.dragged

ドラッグ中。わずかに浮き、回転、影。並べ替え UI 等で使用。
scale: 1.02 / rotate: 1deg
shadow: 0 12px 32px rgba(15,23,42,0.18)
opacity: 0.92 / cursor: grabbing

state.selected

選択中。地域選択、フィルタ、ラジオ。soft（淡塗り＋枠）と strong（ベタ）の 2 段。
京都府 ✓
soft bg: rgba(38, 183, 188,0.10)
border: 2px #26B7BC / fg: #26B7BC
strong bg: #26B7BC / fg: #fff
