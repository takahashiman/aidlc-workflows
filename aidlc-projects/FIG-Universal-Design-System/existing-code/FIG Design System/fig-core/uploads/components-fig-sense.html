FIG Sense — コンポーネントの「らしさ」

FIGデザインシステムの「スピード・慈愛・俯瞰」を、角丸・アイソレーション・最小サイズの セマンティックトークンとしてUI部品に継承するためのショーケース。 プリミティブを直接参照せず、意味的ロール（--radius-cta 等）を経由します。

1. 設計原則

FIGブランドDNA（Rounded-X M+ / 源ノ角ゴシック / ターコイズ・ブルー）から、UI部品の角丸ロールを導出します。

Rounded DNA

主要 CTA は Pill 形状に統一。FIGロゴの円弧 DNA を最も特徴的に継承するロール。

慈愛のやわらかさ

カードは 16px の中庸な角丸で「読みやすさ・親しみ」を担保。鋭すぎず冗長すぎない。

俯瞰の信頼感

Secondary は 16px で硬さを残し、Primary との優先順位を視覚化する。

アイソレーション原則

ロゴマニュアル準拠：上左右 0.3A / 下 0.25A の余白をブランド要素・CTA 周辺に確保する。

2. Radius ロール（意味的角丸）

プリミティブの --radius-{sm,md,lg,xl,2xl,pill} をUIロールに束ねた意味的トークン。コンポーネントは必ずこの層を経由します。

CTA
--radius-cta (pill)

--radius-button (16)
Card
--radius-card (16)

--radius-badge (6)

3. Button — FIGセンスの核

主要 CTA を Pill 形状に統一し、ブランドの語気を担う。Secondary は中庸な 16px で信頼感を残す。

4. Card — 慈愛のやわらかさ

Default は --radius-card（16px）。Hero は --radius-card-hero（28px）で所有感を表現。

標準カード
16px 角丸 / card 影 / 標準パディング
密集カード
12px 角丸 / リスト密度向け
ヒーローカード
28px 角丸 / ブランドティント影 / 定期券・所有面

5. Isolation Zone — ブランド要素の周囲を守る

ロゴマニュアル準拠：上左右 0.3A / 下 0.25A の余白ゾーン。 ブランドマーク・主要 CTA・通知の周囲には他要素を寄せず、意味を弱めない。 --isolation-A を上書きすると比率に従って自動再計算されます。

FIG
.isolation-zone · --isolation-A: 64px → around 19.2px / bottom 16px
定期券
.isolation-zone.isolation-zone--lg · --isolation-A-lg (80px) 大型ヒーロー用

6. 最小サイズ（Touch Target）

視覚サイズが小さくても、タップ可能領域は必ず --a11y-touch-target（44px）以上を確保。 Primary CTA は --interactive-primary-min-height（48px）。

Primary 48px / Secondary 44px / Compact 36px
※ Compact は padding か --a11y-hit-area-expand で 44px の hit area を確保。
