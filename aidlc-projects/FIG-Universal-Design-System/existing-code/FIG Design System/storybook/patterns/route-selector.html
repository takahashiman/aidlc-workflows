Accessibility

✅
Quick Filter： role="group" + 各 chip aria-pressed。
✅
Selector List： role="search" + 結果 role="listbox" + 行 role="option"。
✅
IME 対応： composition 中はフィルタ発火しない。lang="ja" 明示。
✅
検索件数 SR 通知： aria-live="polite" で「3 件の路線が見つかりました」。
✅
ショートカット： / で検索フォーカス、Escape でクリア。
