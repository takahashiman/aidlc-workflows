Accessibility

✅
role="dialog" + aria-modal="true" + aria-label="メインメニュー"。フォーカスを drawer 内にトラップ。
✅
初期フォーカス： 閉じるボタンに移動。Esc / 背面タップで閉じる。
✅
セクション見出しは <h3>。drawer 内の見出し階層は h2 (アバター名) → h3 (グループ)。
✅
Region tag： 視覚で省略している「府」を aria-label="現在の地域: 京都府" で補う。
✅
Logout： アクションは <button>。確認モーダルを必ず挟む（誤タップ防止）。
