Status Pill

バス便の運行状態を一目で伝える。色だけで判断させず、必ず Icon + Label のペアで構成。

States

Pill は表示専用（非インタラクティブ）。状態は data 駆動でのみ変化する。

TriggerBehavior
ETA > 定刻 +3 分normal → delayed、旧 ETA に取消線
他便で遅延発生normal → possible-delay
運休フラグdelayed / normal → suspended
状態クロスフェード--motion-tab-switch (200ms)、色 hue ジャンプ禁止

Accessibility

✅
Icon + Label 必須： 色覚多様性 + 走査速度の両立。アイコン単独 / 文字単独は禁止。
✅
role="status" aria-live="polite" を付与し、状態変化を SR で再アナウンス。
✅
Contrast 検証済： 全 5 状態で fg/bg コントラスト 4.5:1 以上（AA 合格）。
⚠
Possible-delay vs Delayed： amber と orange の hue は色覚多様性で識別が難しいため、Icon shape（triangle vs clock）で明示的に区別。

Motion

状態変化のみ motion を持つ。常時アニメーションは禁止（情報の意味が変わって見える）。

Status crossfade

禁止： Pill の breathing / pulse / 無限点滅。情報の真剣味を損なう。
