Variants

5 つのバリアント。1 画面 1 Primary が原則。Tertiary はキャンセル系、Destructive は破壊的操作のみ。

Primary
主要 CTA（1 画面 1 つ）
--color-surface-brand · elev-floating
Secondary
補助 CTA
--color-surface-inverse · elev-card
Tertiary (Ghost)
テキスト型、低階層
transparent · text-brand
Destructive
破壊的操作のみ
--color-feedback-error-strong
Icon-only

States

すべての state は --state-* token を経由する。:focus-visible で --a11y-focus-ring を使用。

StatePreviewTokenBehavior
default —標準
hover --state-hover-overlay半透明オーバーレイ
pressed --state-pressed-scalescale(0.98)
focused --a11y-focus-ringキーボード操作時のみ
disabled --state-disabled-opacityopacity 0.40 + aria-disabled
loading —aria-busy="true"

Spacing

Padding / 角丸 / hit area は --space-* / --radius-* / --a11y-touch-target から取得。生 px 禁止。

PropertyTokenValue
padding (vertical) --space-3 12px
padding (horizontal) --space-5 20px
gap (icon ↔ label) --space-2 8px
radius --radius-xl 20px
min-height --a11y-touch-target48px

Accessibility

WCAG 2.5.5 (Target Size) / 2.4.7 (Focus Visible) / 1.4.3 (Contrast) を満たす。

✅
Hit area： 48×48 px（WCAG 推奨 44×44 以上を確保）
✅
Focus ring： :focus-visible で --a11y-focus-ring を 2px outline + 2px offset で描画。:focus 全消し禁止。
✅
Disabled： disabled 属性 + aria-disabled="true" を併記。属性 disabled だけだとフォーカスが当たらない場面がある。
✅
Loading： aria-busy="true" を付与、スピナーには aria-hidden="true"。
✅
Icon-only： aria-label 必須。lucide アイコン単独では意味伝達不可。
✅
Contrast： Primary (brand on white) = 5.2:1, Destructive (red 600 on white) = 4.6:1。両者 AA 合格。

Motion

すべての遷移は motion token 経由。cubic-bezier() 直書き禁止。

Hover
Background fade
--motion-hover (160ms / standard)
Press
scale(0.98) feedback
--motion-press (80ms / standard)
Enter (loading → result)
Fade + 8px translate
--motion-enter (280ms / decelerate)
禁止： bounce / spring 系、durations 直書き (transition: 200ms ease)、independent cubic-bezier() 関数。
