Accessibility

✅
role="tablist" をコンテナに、role="tab" を各セグメントに付与。
✅
aria-selected="true|false" + aria-controls={panelId}、対応する panel に role="tabpanel"。
✅
キーボード： ← / → でフォーカス移動、Home / End で端ジャンプ、Enter / Space で activate。
✅
Touch target 44×44。コンテナ padding と segment min-height で確保。

Motion

タブ切替は単一の motion token で統一。コンポーネントごとに ease を変えない。

Tab switch
クリックで切替
--motion-tab-switch (200ms / standard)
禁止： 横スライド（コンテンツの遷移と意味が衝突）、bounce、scale を大きく動かす運動（0.95 → 1.0 程度に抑える）。
