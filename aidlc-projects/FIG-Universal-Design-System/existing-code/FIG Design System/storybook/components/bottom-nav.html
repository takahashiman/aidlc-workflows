Accessibility

✅
<nav aria-label> で囲む。各アイテムは <button> または <a>。
✅
aria-current="page" をアクティブに付与（aria-selected ではない）。
✅
Badge は装飾扱いしない： aria-label="未読 3 件" 等の可読文を付与。
✅
Compact variant でも aria-label でラベル相当の情報を提供（視覚的にアイコンしかなくても識別可能に）。
✅
全アイテム 44×44 以上： 56px item-height + 適切な padding で確保。

Motion

タブ切替は --motion-tab-switch で統一。激しい scale 変化や bounce は禁止。

禁止： モーダルがマウントされている間も Bottom Nav が動く（モーダル時は背後に固定 or 非表示）、横スクロール、無限振動の badge。
