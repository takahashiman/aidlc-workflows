TRANSPORT DOMAIN

status / route / vehicle

交通ドメイン固有のセマンティック。プロジェクトの差別化価値となるトークン群で、UI のどこにバスや電車、運行状態が登場しても一貫した色で表現します。

status.* — 運行状態
通常運行
status.onTime
fg #047857 · bg #ECFDF5
遅延なく定時運行中。
遅延の可能性
status.delayRisk
fg #B45309 · bg #FFFBEB
混雑等で遅延の見込み。確定ではない。
約15分遅延
status.delayed
fg #B91C1C · bg #FEF2F2
遅延確定。分数を併記。
運休
status.suspended
fg #B91C1C · bg #FEF2F2
運休。理由メッセージとペアで表示。
通過済
status.passed
fg #64748B · bg #F1F5F9
該当便は通過済み。非活性扱い。
