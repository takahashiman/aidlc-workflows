Elevation

box-shadow と z-index を対にした 6 段階の浮き上がり。コンポーネントは生のシャドウ値を直書きせず、必ずこのラダーを経由します。

elevation
0
フラット。背景面、入力フィールド、リストセル。
shadow: none z-index: 0 (--z-base)
elevation
1
静止状態のカード、低位ボタン、リスト行の浮き。
shadow: 0 1px 2px / 0 1px 3px z-index: 1 (--z-raised)
elevation
2
ホバー状態のカード、プルダウン、サイドバー。
shadow: 0 4px 12px / 0 2px 4px z-index: 20 (--z-dropdown)
elevation
3
FAB（浮遊ボタン）、スナックバー、固定 CTA。
shadow: 0 8px 30px rgba(0,0,0,.12) z-index: 40 (--z-fab)
elevation
4
ダイアログ、モーダル、ボトムシート、定期券カード。
shadow: 0 20px 50px -8px (+補助) z-index: 60 (--z-modal)
elevation
5
トースト、最前面の通知、エラーフラッシュ。
shadow: 0 25px 50px -12px (+補助) z-index: 80 (--z-toast)

重なり順デモ

同じステージに 0〜5 を並べて、z-index の上下関係も確認できます。

0 base
1 raised
2 dropdown
3 fab
4 modal
5 toast
