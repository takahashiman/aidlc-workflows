通知設定

運行情報のプッシュ通知、メール通知、SMS 通知の頻度と種類を細かく制御できます。緊急時の通知は無効化できません。

  • 運行遅延：即時 / 5 分以上 / 15 分以上
  • 路線変更：当日 / 前日 / 1 週間前
  • 運休：自動通知
プライバシー

位置情報の利用範囲、データ収集の同意、第三者提供の可否を管理できます。

支払い情報

登録済みのカード、IC カード、ポイント残高の管理。

FAQ スタイル
Q IC カードはいつから使えますか?

Suica / PASMO / ICOCA など全国 10 種の交通系 IC カードに対応しています。初回ご利用時にアプリ内で IC カード番号を登録してください。

Q 定期券は購入できますか?

マイページ > 定期券 > 新規購入 から、区間・期間を指定して購入できます。1 ヶ月 / 3 ヶ月 / 6 ヶ月の 3 種類。

Q 運休時の払い戻しはどうなりますか?

運休が確定した時点で自動的に運賃が返金されます（IC カード残高 / ポイント / クレジットカードへの返金）。手続きは不要です。
